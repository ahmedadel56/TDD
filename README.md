# TDD project

- Write a test.
- Run the test; it fails.
- Write code.
- Run the test; it passes. If it fails go back to step 3.

## Written With
- Ruby

## Getting Started
Before we start, make sure you fit the prerequisites

### Setup
- git clone 
  
### Install
- `bundle install`

### Usage
- `ruby main.rb` from Terminal

### Run linters tests
- `rubocop`
